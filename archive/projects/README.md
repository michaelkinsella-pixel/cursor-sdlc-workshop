# Archived workshop projects (`archive/projects/`)

Active workshop scaffolding stays at `projects/project_template/` and `projects/README.md`. **Retired participant trees** from past cohorts live here as subfolders (for example `melissagannes/`, `bryson-droid/`).

Do **not** commit `node_modules`, build output (`dist`, `.next`, etc.), environment files (`.env`), or other generated artifacts. New work under `projects/<github-username>/` should still include a `.gitignore` (the template provides one under `base_mvp/`).
